# 19childja

WordPressの公式テーマ、Twenty Nineteenの子テーマです。

テーマサポートの実例として公開しています。

## Install
dev/ で `npm install`

## Use

### 設定

dev/_config.scss でテーマカラーを設定しています。

### 書き出し

`npm run build` でSassのコンパイル

`npm run watch` でSassの監視

`npm run color_config`で色の書き出し（開発途中）
